3. Pipe

O Pipe é utilizado para formatar a forma como os dados aparecem na tela, ex:

  • Data: nome_da_variavel | date: "dd/MM/yyyy"
  • String: variavel | UpperCasePipe ou LowerCasePipe
  • Unidade de Moedas: variavel | CurrencyPipe
  • clique aqui para ver mais sobre pipes;

É possível criar seu próprio Pipe, basta criar usando o comando ng generate pipe nome-do-pipe.

Exemplo:

Criei no meu projeto um pipe para multiplicar, usei o comando: ng generate pipe multiplica-por.
