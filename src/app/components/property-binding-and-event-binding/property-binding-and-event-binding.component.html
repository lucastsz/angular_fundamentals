4. Property Binding (PB) e Event Binding (EB)

  1. Property Binding

    O PB permite que você use variáveis como propriedades dos componentes HTML dentro do componente, exemplo: Criar uma variavel no arquivo TypeScript do componente chamada urlImage, após a criação, será possível usar essa variável como propriedade de alguma tag ou elemento HTML.

    Estanciei a variável no arquivo .ts do meu componente assim: urlImage: string = "https://source.unsplash.com/random/300×300/".
