2. Interpolação

A interpolação serve para mostrar os dados(variáveis) que estão na classe do componente no template (HTML) do componente, através do uso do: {{ "{{nome_da_variavel}}" }}.

Para utilizar basta criar a variavel no arquivo nome-do-componente.ts e instanciar no arquivo nome-do-componente.html usando o: {{ "{{nome_da_variavel}}" }}.

Exemplo:

Foi instanciada uma variável com o nome "variável interpolada" no componente do meu projeto chamado interpolacao.ts, agora será exibida aqui através da interpolação: {{ interpolacao }}

  • Arquivo (interpolacao-texto.ts): interpolacao: string = "variável interpolada"
  • Arquivo (interpolacao-texto.html): {{ "{{interpolacao}}" }}.
