Introdução ao Angular

1. Primeiros passos com o Angular

  1. Para criar um projeto Angular, digite no terminal o comando: ng new nome-do-projeto e pressione Enter.
  2. Durante a criação do projeto, o angular perguntará se deseja adicionar o Angular routing digite Y e pressione Enter.
  3. Depois perguntará qual o formato de estilização que você gostaria de usar, pressione enter para adicionar o CSS (1º Opção).
  4. O angular é baseado em componentes, cada componente irá ter um arquivo CSS, HTML, TypeScrit e TypeScrit para testes.
  5. Os arquivos raízes da aplicação

1.1. Criar Componentes

Para criar componentes no projeto você deverá digitar no terminal do projeto o seguinte comando:

  • Digite: ng generate component nome-do-componente e pressione Enter.
  • Para organizar melhor os componentes e deixar mais limpa a aplicação, você poderá criar uma pasta components para organizar melhor os componentes na aplicação). Nesse caso você poderá digitar esse código em vez do outro:
    • ng generate component components/nome-do-componente

Para Exibir o seu componente criado no app você terá que estanciar o componente no arquivo app.component.html. Perceba que assim que você cria um novo projeto esse arquivo app.component.html vem preenchido como padrão do angular, você pode selecionar tudo e deletar.

1.2 Estanciando um componente no projeto raiz

Abra o arquivo app.component.html e digite: < app-nome-do-componente > </app-nome-do-componente > .

Caso não tenha iniciado sua aplicação, digite no terminal: ng serve ou npm start caso tenha o npm instalado na sua máquina.
